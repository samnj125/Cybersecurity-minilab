# Cybersecurity MiniLab 🛡️

## 📌 Overview
This is a Python-based cybersecurity mini lab that includes a set of tools designed to improve digital security awareness and practical skills.

The toolkit contains:

- 🔐 Password Strength Checker
- 🔎 Hash Generator (MD5 & SHA256)
- 🔒 Encryption Tool (Fernet-based encryption)
- 📡 Port Scanner

Each tool serves a different purpose in understanding basic cybersecurity concepts.

---

## 🔐 Password Strength Checker
Analyzes a password using a point-based system and provides feedback on its strength based on:
- Length
- Uppercase characters
- Numbers
- Special characters

---

## 🔎 Hash Generator
Generates a hashed value using:
- MD5
- SHA256

This demonstrates how hashing works in data integrity and password storage.

---

## 🔒 Encryption Tool
Encrypts a message using Fernet encryption and generates a key required for decryption.

This demonstrates symmetric encryption concepts.

---

## 📡 Port Scanner
Scans a target IP address and identifies open ports within a specified range and gives a report on the scan. 

This demonstrates basic network scanning and socket programming.

---

## ▶️ How To Run

1. Install Python (3.x recommended)

2. Install required dependency:
   pip install cryptography

3. Run the program:
   python cyberlab.py

4. Follow the on-screen menu to select a tool.

---

## ⚠️ Ethical Use Warning
These tools are intended for educational purposes only.

Only use the port scanner and other tools on:
- Your own systems
- Lab environments
- Systems where you have explicit permission

Unauthorized scanning or misuse may be illegal.
